# 🎓 Smart College Assistant - Frontend

Next.js frontend for the Smart College Assistant system.

## 🌐 Live Demo
- **Dashboard:** https://smart-college-assistant-jbld-idgjju5kr.vercel.app/dashboard
- **Chatbot:** https://smart-college-assistant-jbld-idgjju5kr.vercel.app

## 🔧 Backend Repository
https://github.com/Heenajindal03/smart-college-assistant

## ✨ Features
- Login with credentials or Face Recognition
- Student Dashboard — attendance, grades, analytics, chatbot
- Faculty Dashboard — manage students, grades, announcements
- Monthly Planner & Reminders
- Leave Application System
- Charts & Analytics
- Smart Attendance Predictor
- Internal Marks Calculator
- Dark/Light Mode

## 🛠️ Tech Stack
- Next.js 16, React 19, TypeScript
- Tailwind CSS, Recharts, Lucide React
